Mobile Application (React/React Native) Developer

Years of experience: 5+

Location: Any

Job Short Summary

  • 5+ Years
  • Remote
  • React native, Java script
Send your CV

Contact us at hr@mongrov.com for any inquiries.

Job Description:

We are looking for talented Mobile application developer who is keen to join our engineering team to build up a great application. You will be taking a developer role in building React / React Native / JavaScript applications.

The position requires a developer with experience and expertise building modern mobile web apps using the latest front-end technology. At Mongrov, we are using JavaScript with React/React Native to build our next generation platform.

Roles and Responsibilities:

  • Understand the product requirements
  • Develop new user-facing features
  • Write reusable code and build libraries for future use
  • Ensure the technical feasibility of UI/UX designs
  • Optimize application for maximum speed and scalability
  • Collaborate with Product Designers, Product Managers, and Backend Engineers to deliver compelling user-facing products

Skills Required:

  • Expert knowledge of React, React Native, JavaScript, HTML, CSS and related tools/packages
  • Excellent troubleshooting skills
  • Good in git and git workflow
